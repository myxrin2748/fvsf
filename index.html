📖 Cara Pakai

1
Buka Instagram → Settings → Your activity → Download your information → Request download dalam format JSON
2
Tunggu email dari Instagram, lalu download dan extract file ZIP-nya
3
Cari file following.json dan followers_1.json di dalam folder connections/followers_and_following/
4
Upload kedua file ke aplikasi ini, lalu klik Cek Sekarang
